# Task Manager API

This project is a simple REST API built using Python and FastAPI that allows users to manage tasks.

## Overview

The API provides endpoints that allow users to create tasks and view a list of existing tasks. It demonstrates basic backend development concepts such as API design and HTTP request handling.

## Features

- Create tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- REST API endpoints

## Technologies Used

- Python
- FastAPI
- REST API

## Purpose

This project was created to practice backend development using Python and FastAPI. It demonstrates how APIs can be built to manage data and handle client requests.
